# VocabQuiz

大学受験向け英単語を、スペル入力形式で学習できるWebクイズアプリ

## 特徴
- スペル入力型の英単語クイズ
- 難易度別出題（★ / ★★ / ★★★）
- 正答数・正答率のリアルタイム表示
- 間違えた単語の復習機能

## 使い方
1. クイズを開始
2. 英単語のスペルを入力
3. 結果を確認し、復習する

## 著作権に関する注記
本アプリの単語リスト作成にあたり、収録語彙の選定の参考として『ターゲット1900』（旺文社）を参照しました。
ただし、各英単語に対応する日本語の意味はすべて独自に作成したものであり、同書の訳語・解説を転載・複製したものではありません。英単語そのものは著作権の保護対象にはあたらないと解されており、本アプリは著作権侵害を意図したものではありません。
もし問題がある場合はIssueにてご連絡ください。
